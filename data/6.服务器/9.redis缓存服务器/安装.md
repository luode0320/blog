# 拉取redis

```sh
docker run -d \
--restart=always \
--name redis \
-p 6379:6379 \
luode0320/redis:latest \
--requirepass Ld588588 \
--bind 0.0.0.0

# 不设置密码
docker run -d \
--restart=always \
--name redis \
-p 56379:6379 \
luode0320/redis:latest \
--protected-mode no \
--bind 0.0.0.0
```

测试:

```sh
# 设置一个key
docker exec -it redis redis-cli SET key "value"
# 查询
docker exec -it redis redis-cli GET key 
```
