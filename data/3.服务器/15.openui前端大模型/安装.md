# 简介

这是一个2024.5.18构建的[wandb/openui](https://github.com/wandb/openui)

# 安装

```sh
docker pull luode0320/openui:latest
```

# 启动

- 这个版本需要使用openai

```sh
docker run -d \
-p 6000:7878 \
--restart=always  \
--name openui \
-e OPENAI_API_KEY="open ai key" \
luode0320/openui:latest
```

# 访问

```
http://localhost:6000
```
