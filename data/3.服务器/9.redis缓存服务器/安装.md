# 拉取redis

```sh
docker pull luode0320/redis:latest
```

```sh
docker run -d \
--restart=always \
--name redis \
-p 6379:6379 \
luode0320/redis:latest \
--requirepass 123456
```

# 拉取redis可视化工具

这是一个redis的web网页端可视化工具

```sh
docker pull luode0320/redis-web:latest
```

```sh
docker run  -d \

--name redis-web \
```
